**Muslim Companion** เป็นแอปพลิเคชัน Android ขนาดเล็กและเชื่อถือได้  
ออกแบบมาเพื่อช่วยอำนวยความสะดวกแก่ชาวมุสลิมในการปฏิบัติศาสนกิจประจำวัน

## คุณสมบัติ
- แสดงทิศกิบละห์อย่างแม่นยำด้วย GPS และเข็มทิศของอุปกรณ์  
- แสดงเวลาละหมาดตามตำแหน่งที่ตั้งปัจจุบัน  
- แจ้งเตือนเวลาละหมาด พร้อมเสียงอาซานหรือระบบสั่น (สามารถปิดเสียงได้)  
- รองรับหลายภาษา: อังกฤษ ไทย และอาหรับ  
- ดีไซน์เรียบง่าย สะอาดตา และใช้งานสะดวก  

## ข้อมูลแอปพลิเคชัน
- **แพลตฟอร์ม:** Android  
- **ประเภทไฟล์:** APK  
- **ขนาดไฟล์:** ประมาณ 7.38 MB  
- **ค่าใช้จ่าย:** ใช้งานฟรีสำหรับการใช้งานส่วนบุคคล  

## ดาวน์โหลดและการติดตั้ง
แอปพลิเคชันนี้เผยแพร่โดยตรงจากผู้พัฒนาผ่าน **GitHub Releases**

1. ดาวน์โหลดไฟล์ APK เวอร์ชันล่าสุดจากหน้า **Releases**  
2. เปิดการอนุญาต **ติดตั้งแอปจากแหล่งที่ไม่รู้จัก** บนอุปกรณ์ Android  
3. ติดตั้งไฟล์ APK และเริ่มใช้งานแอป  

** การติดตั้งเป็นทางเลือก ขึ้นอยู่กับดุลยพินิจของผู้ใช้

## สิทธิ์ที่แอปต้องการ
- **ตำแหน่งที่ตั้ง** (ใช้สำหรับการคำนวณทิศกิบละห์และเวลาละหมาด)  
- **การแจ้งเตือน** (ใช้สำหรับแจ้งเตือนเวลาละหมาด)
